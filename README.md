# Personal Portfolio Website

## Project Overview
This is my **Personal Portfolio Website** developed using **HTML** and **CSS** to showcase my skills, projects, and contact information in a clean and interactive way. The portfolio reflects my passion for **web development**, creativity, and modern UI design.

The website is designed with smooth animations, responsive sections, and an attractive user interface to provide a better user experience.

## Features
● Responsive and visually appealing design  
● Smooth animations for an interactive experience  
● Home, About, Skills, Projects, and Contact sections  
● Technical and Professional Skills showcase with animated progress bars  
● Social media integration (Instagram, LinkedIn, GitHub, WhatsApp)  
● Contact form for user interaction using Formspree  
● Resume viewing option  

##  Technologies Used
- HTML5  
- CSS3  
- Boxicons  
- Formspree (for contact form functionality)

##  Project Structure
- **Home Section** → Introduction and social media links  
- **About Section** → Brief introduction about me  
- **Skills Section** → Technical and professional skills with animations  
- **Projects Section** → Showcasing my interests and project details  
- **Contact Section** → Contact details and message form

# LANDING PAGE
GRIT is a smart routine planning web application idea designed to help people build a more scheduled and disciplined lifestyle. The name GRIT represents determination, discipline, consistency, and self-improvement.

This project focuses on organizing daily activities by allowing users to create routines, schedule tasks, set activity timings, alarms, and automatically reschedule missed tasks.

Currently, this project is not fully implemented — it is an outline/prototype showcasing the core idea and planned features for building a productivity-focused daily planner.
## Project Flow / Outline
- **Plan Your Day** → Add tasks and routines
- **Save Tasks** → Store planned activities
- **Set Activity Time** → Assign timings to tasks
- **Set Alarm** → Add reminders/ringtones
- **Auto Reschedule** → Automatically shift missed tasks

# Calculator Project
This Calculator Project is a web-based calculator developed to perform basic mathematical operations in a simple and user-friendly interface. The project was created as a learning step to improve understanding of frontend development, DOM manipulation, and JavaScript logic.

Currently, the calculator represents an outline/basic implementation and may be expanded with more advanced features in the future.

## Features Used
## Basic arithmetic operations
● Addition (+)
● Subtraction (-)
● Multiplication (×)
● Division (÷)
● Interactive button-based UI
● Real-time display updates
● Clear/Delete functionality
● Simple and responsive design
## Technologies Used
● HTML5 – Layout structure
● CSS3 – UI styling
● JavaScript – Calculator logic and operations
